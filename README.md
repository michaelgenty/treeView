# treeView

## This project uses angular and the file manager solution from syncfusion


Steps to follow:

1/ 
```bash
npm install
```

2/
```bash
cd node_modules\@syncfusion\ej2-filemanager-node-filesystem
```
```bash
npm install
```

3/
```bash
npm run explore -- C:/yourFolderPath
```


4/
```bash
ng serve
```
